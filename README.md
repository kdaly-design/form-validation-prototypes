# Form Validation Prototypes

Interactive HTML prototypes demonstrating form validation patterns with WellSky Atlas Design System.

## Prototypes

### Full Page Example
A complete payer creation form demonstrating comprehensive form validation:
- Required field validation on blur
- Format validation (email, phone, zip code)
- Error banner with clickable error links
- Accessible ARIA attributes
- Material fill form fields

[View Full Page Example](./full-page-example/index.html)

## Running Locally

No build step required! Just open any `index.html` file in a browser:

```bash
open full-page-example/index.html
```

## Features

- **WellSky Atlas Design Tokens** - Uses official design system tokens
- **Accessible** - Proper ARIA labels and keyboard navigation
- **Responsive** - Mobile, tablet, and desktop layouts
- **Zero Dependencies** - Pure HTML, CSS, and vanilla JavaScript
